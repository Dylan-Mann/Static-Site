Simpel

Klik en klaar! Zo snel als je je server gekocht hebt wordt hij direct voor je gemaakt. Je kan hem gelijk besturen met het mooie multicraft control panel. En als je toch ooit een probleem of opmerking hebt kan je in een paar klikken een support ticket openen.

Betaalbaar

Je krijgt een server met alles er op en er aan voor een lage prijs. Geen opstart kosten, geen achteraf kosten. Je betaald je maandelijkse kosten. Niet meer, niet minder. Je kan zelfs een subdomein van ons korte sshmc.in domein krijgen, gratis. Zo betaal je voor een één gigabyte server bij ons maar zes euro per maand!

Beresterk

Je server draait op een quad-core processor met hyper-threading technology. Het is daarnaast ook nog een intel-i7 processor, één van de beste van het moment. Ook zal je server gebruik maken van de laatste versie van Oracle Java Virtual machine. Daarmee is hij zo veilig en snel als mogelijk. De ram die je krijgt, is totaal van jou. Wij zullen niet de ram die je niet gebruikt aan andere verkopen, of er eigen programma's op draaien. Zelfs als je server uit staat ben jij de enige die de ram kan gebruiken. Zo weten we zeker dan als jij je server opstart, jij de ram kan gebruiken die voor jou bedoeld is. Vertrouw op onze hard- en software kwaliteiten, dan kan jij je bezig houden met de plugins en de mensen.

support

Wanneer jij een support ticket maakt, wordt ons hele team gewaarschuwd. Dan is het bij ons meestal vechten om wie jou ticket mag beantwoorden. Dus reken maar dat je super snel antwoord krijgt! Wij geven ook basis plugin hulp. Voor een klein bedrag krijg je volledige plugin hulp. Jouw mannetje installeert dan elke plugin die je wilt!

Proefperiode

Als je nog twijfelt moet je zeker een gratis proeftijd nemen. Hiermee kan je drie dagen lang onze service testen, als je het niets vindt kost het je helemaal niets. Als je verder wilt kan je makkelijk je proef omzetten in een betaald pakket. Krijg hier je proefperiode, kost nog geen minuut!
